For database scaffolding using the new recomended tool by LINQDB  https://www.nuget.org/packages/linq2db.cli

If you dont have the tool open command line for the project and run 
	dotnet tool install --global linq2db.cli --version 5.2.0

To regenerate the database model, run the generate_and_run_scaffold.ps1 script
You also need to set up the env variables for your machine, sent picture in discord
